# Testing1

A simple Kotlin application built with Gradle.  
This project serves as a testing ground for experimenting with Kotlin features, Gradle build automation, and GitHub workflows.

## Features

- Kotlin-based console application
- Gradle build system (Kotlin DSL)
- Structured package: `io.umsf.pimonovmykyta.testing`
- Pre-configured issue templates for contributions

## ️ Getting Started

### Prerequisites

- **JDK 11** or later  
- **Gradle** (optional – wrapper included)

### Installation & Running

1. **Clone the repository**
   ```bash
   git clone https://github.com/PimonovNI/testing1.git
   cd testing1
   ```

2. **Build the project**
   ```bash
   ./gradlew build   # On Unix/macOS
   gradlew.bat build # On Windows
   ```

3. **Run the application**
   ```bash
   ./gradlew run
   ```
